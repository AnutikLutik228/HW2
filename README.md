Проект на Python для работы с рецептами, ингредиентами, диетическими рецептами и списком покупок.

В этом проекте реализована небольшая система для управления рецептами с использованием объектно-ориентированного программирования.

Программа позволяет:
- создавать ингредиенты с названием, количеством и единицей измерения
- объединять ингредиенты в рецепт
- добавлять новые ингредиенты в уже созданный рецепт
- увеличивать или уменьшать количество ингредиентов в рецепте
- создавать диетические рецепты с указанием типа диеты
- формировать общий список покупок на основе нескольких рецептов.



Классы:
Ingredient - Хранит информацию об ингредиенте: название, количество и единицу измерения |
Recipe - Описывает рецепт и позволяет работать со списком ингредиентов |
DietaryRecipe - Расширяет обычный рецепт и добавляет тип диеты |
ShoppingList - Создает общий список покупок из добавленных рецептов |


Установка:
Сначала нужно склонировать репозиторий:
git clone https://github.com/AnutikLutik228/HW2.git

Перейти в папку проекта:
cd HW2

Установить зависимости:
python3 -m pip install -r requirements.txt


Запуск программы:

Запустить основной файл можно командой: python3 main.py

Запуск тестов:
Для проверки работы программы нужно запустить тесты:
python3 -m pytest test_recipes.py


Если программа работает правильно, в терминале появится сообщение о пройденных тестах, например:
19 passed
