Этот проект сделан для работы с рецептами.
В нём есть классы Ingredient, Recipe, DietaryRecipe и ShoppingList.
Ingredient хранит название продукта, количество и единицу измерения.
Recipe хранит рецепт и позволяет добавлять ингредиенты и менять их количество.
DietaryRecipe хранит рецепт и тип диеты.
ShoppingList собирает продукты из нескольких рецептов в один список покупок.
Для проверки работы программы используются тесты на pytest.

Использование:
python main.py

pytest

Установка:
git clone <ссылка на репозиторий>
cd HW2
pip install -r requirements.txt
pytest
